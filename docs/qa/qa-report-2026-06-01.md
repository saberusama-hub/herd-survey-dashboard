The link crawler still flags `*.parquet` URLs as `ERR_ABORTED`. This is a

by DuckDB-WASM via the OPFS-backed file-handle layer, not by the page's

hit directly; the crawler aborts because the response isn't an HTML
document. Documented as expected behaviour, not a failure.
